DESCRIPTION
===========

Display a table of registered tasks. The table columns show

 1. The registered name of the tasks. This key can be used to run specific tasks
 2. The name of the monitor procedure that the task will be run from 
 3. A description of the actual purposes of the task
 4. The name of the script implmenting the task
 5. The language of the script. Supported languages are: *bash*, *Tcl* and *Python3*
